# –¶–∏–∫–ª—ã

–í —Ä–∞–º–∫–∞—Ö –¥–∞–Ω–Ω–æ–≥–æ —É—Ä–æ–∫–∞ –∏–∑—É—á–∏–ª–∏ –æ—Å–Ω–æ–≤–Ω—ã–µ —Ü–∏–∫–ª—ã, —Ç–∞–∫–∏–µ –∫–∞–∫ `for`, `while`, `for of`, `for in`:


# –ú–∞—Å—Å–∏–≤—ã

–¢–∞–∫–∂–µ, —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–ª–∏ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—é –º–∞—Å—Å–∏–≤–æ–≤, –æ–±—Ä–∞—â–µ–Ω–∏–µ –ø–æ –∏–Ω–¥–µ–∫—Å–∞–º –∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ –º–µ—Ç–æ–¥–æ–≤ –º–∞—Å—Å–∏–≤–æ–≤:

```js

console.log(arr[2]) // "here"

// Reverse
```
